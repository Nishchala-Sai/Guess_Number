# Number Guessing Game

A number guessing game which uses the concepts of functions and DOM in JS. The CSS styling is made to fit a cute, bubbly concept.

## How to Play

1. Enter a number between 1 and 100
2. Click **Submit Guess**
3. The game tells you if your guess is too high or too low
4. Keep guessing until you find the correct number
5. Your attempt count is tracked — try to beat your best score!

## Features

- Random number generation between 1 and 100
- Too high / too low feedback on each guess
- Attempt counter
- Input validation
- New number generated automatically after a correct guess

## Tech Stack

- HTML
- CSS (custom fonts, pixel cat decorations)
- JavaScript (DOM manipulation, functions)
